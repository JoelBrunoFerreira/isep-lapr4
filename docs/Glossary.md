
# Jobs4u - Glossary

| **Term**                         | **Description**                                                                                                                                                                      |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ANTLR                            | ANTLR (ANother Tool for Language Recognition) is a powerful parser generator for reading, processing, executing, or translating structured text or binary files.                     |
| Applications                     | Formal documents submitted by an individual to apply for a specific job or position within a company or organization by email.                                                       |
| Applications Email Bot           | A bot responsible for processing emails received as job applications and producing files adapted for integration into the system.                                                    | 
| Applications File Bot            | A bot responsible for processing files produced by the Applications Email Bot and integrating candidates and applications into the system.                                           | 
| Automated Process                | Use of technology to streamline and expedite various stages of candidate selection for job openings, such as interviews or verification of application requirements.                 | 
| Backoffice                       | The administrative area of a company where tasks such as customer management and application processing are handled.                                                                 | 
| Candidate                        | Individual who has been identified and selected by a company or organization as a potential employee for a specific position.                                                        | 
| Client Manager                   | Refers to Customer Manager.                                                                                                                                                          | 
| Clients                          | Companies or entities interested in recruiting human resources.                                                                                                                      | 
| Customer Manager                 | A company employee responsible for managing customer entities, registering job offers and other aspects, namely managing the entire candidate selection process.                     | 
| Email Application                | Application submitted by candidates for job openings by email.                                                                                                                       |
| Evaluation Rules                 | Criteria used to evaluate candidates' answers during interviews and determine their grades.                                                                                          | 
| Interview Model                  | A sequence of pre-defined questions used to register the answers of candidates during interviews.                                                                                    | 
| Job Offer                        | A vacant position within a company or organization for which the employer is actively seeking to hire a qualified individual.                                                        | 
| Job Opening (System Application) | System app job offer that includes the job reference, title/function, contract type, mode, address, company, number of vacancies, description and requirements for the position.     | 
| Job Vacancies                    | Same as Job Opening (System Application).                                                                                                                                            | 
| Jobs4U                           | A company specialized in talent acquisition, providing recruitment services for job positions in its clients.                                                                        | 
| Language Engineer                | A software engineer responsible for designing and implementing modules that automate processing of job interviews and application requirements using language processing techniques. | 
| Operator                         | A company employee whose main responsibility is to monitor the automatic process that registers applications for job offers.                                                         | 
| Plugin                           | Software component or module that adds specific functionality to a larger software application.                                                                                      |
| Recruitment Process (RP)         | Recruitment process for a job opening that follows a sequence of phases: application, resume screen, interviews, analysis and result.                                                | 
| RP Analysis Phase                | During this phase, the applications are analyzed (using all available information like interviews and curriculum) and candidates are ranked.                                         | 
| RP Application Phase             | During this phase candidates submit their applications.                                                                                                                              | 
| RP Interviews Phase              | During this phase, accepted candidates may be interviewed. Results for the interviews are registered for further analysis.                                                           | 
| RP Result Phase                  | In this phase, candidates as well as customers are notified of the final result.                                                                                                     | 
| RP Screening Phase               | In this phase, applications are verified against a set of requirements. Applications that do not meet mandatory requirements are rejected.                                           | 
| System Administrator             | Person responsible for managing customer entities as well as the company’s employees.                                                                                                | 
| System Users                     | Entity that is able to access and use the Josb4U Application.                                                                                                                        | 
| Template Text File               | A file containing requirements or questions to be evaluated during the recruitment process.                                                                                          |
