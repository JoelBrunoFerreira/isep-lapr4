## 5. Implementation

*The following scripts were implemented:*
To build:

    build-all.bat
    build-all.sh

To rebuild:

    rebuild-all.bat
    rebuild-all.sh

To run JUnit Tests:

    run-JUnitTests.bat
    run-JUnitTests.sh

To perform a quick build, after some change/update in pom.xml files:

    quickbuild.bat
    quickbuild.sh

To run the App (running the JAR file):

    run-candidate-app.bat
    run-candidate-app.sh

    run-customer-app.bat
    run-customer-app.sh
